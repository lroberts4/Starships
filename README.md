# StarshipTest
SWStarships Test:

Purpose:
    This console project is built to evaluate the total number of stops a Starship might need to make in order to travel
a certain distance.

Required Packages and Dependencies:
    SWapiCSharip: helper library for https://swapi.co/
    This project was written using the .NET Framework 4.7.2

To run the console application:
    Run the executable file which is located in the /../Starships/bin/Debug/SWStarships.exe
    The application will prompt for the user to enter the desired distance.
    Once the stops given the distance has been calculated the result will be displayed.
    The application will prompt the user if they want to continue entering input or exit.
